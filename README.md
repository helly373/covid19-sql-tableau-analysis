# COVID-19 Global Data Analysis (SQL Server + Tableau)

This project performs an end-to-end analysis of global COVID-19 data using **SQL Server** for data processing and **Tableau** for interactive visualization.  
It includes raw datasets, the SQL script used for analysis, the Tableau dashboard, and exported workbook files.

1. Import the `.xlsx` files into SQL Server.  
2. Run the SQL script:  
   **`COVID-19 Analysis With SQL.sql`**  
3. Connect Tableau to SQL Server **or** use the `.hyper` files included.  
4. Open the dashboard using:
   - The PDF copy in the repository, or  
   - The Tableau Public link above.
